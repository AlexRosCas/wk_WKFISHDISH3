# Working repository for theme group 2

This repository provides a shared workspace for participants of the theme group
2 *"Spatiotemporal structure and scale"*.

The repository is intended for collaborative development of code, analyses,
notes, figures, and example applications related to the methodological themes
addressed by the subgroup.


## Purpose

Participants may use this repository to:

* develop and share analysis scripts,
* explore workshop case studies,
* compare alternative modelling approaches,
* document methodological questions and findings,
* share figures, results, and example workflows,
* develop recommendations, guidelines, or best practices.

The repository is intended as a collaborative working environment. Content may
evolve rapidly during and after the workshop.


## Suggested structure

```text
data/       Input data and case studies
scripts/    Analysis and modelling scripts
figures/    Figures and visualisations
results/    Intermediate and final results
notes/      Meeting notes and discussion documents
```

Participants are free to adapt this structure as needed.


## Good practice

* Use descriptive file names.
* Document assumptions and analytical choices.
* Include comments in scripts where appropriate.
* Avoid committing large temporary files or derived outputs that can be
  reproduced.
* Clearly indicate the status of analyses (draft, exploratory, final).
